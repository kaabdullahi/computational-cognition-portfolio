# Computational Cognition Portfolio
### Khalid Abdullahi | MClin Student | Institute for Neuroscience, Australia

## Contents
- /honours-thesis — Original honours analysis. Bayesian RM-ANOVA (R) testing 
  scheduling performance across team/separate conditions. Includes the original 
  sampling-distribution simulation (MATLAB, A. Eidels) used to benchmark 
  distance-from-optimality against chance.
- /ddm-analysis — EZ-diffusion reanalysis of honours data: team vs individual 
  decision-making (R). Key finding: teams show significantly worse scheduling 
  optimality than individuals (paired t-test, p = .014, d = 0.85, large effect).

## Research Focus
Computational cognitive psychology — understanding mechanisms of human 
learning, valuation, and choice through formal models (reinforcement 
learning, drift-diffusion), with neurotypical/non-neurotypical 
differences as one application of broader individual-differences research.
